Sistemas de Ordenação

Ordenação de Números

Como funcionam os algoritmos de ordenação?

Bubble Sort: O algoritmo Bubble Sort compara elementos adjacentes e os troca de posição até que a lista esteja ordenada. Ele é simples, mas ineficiente para grandes listas, com complexidade O(n²).

Melhor uso: Pequenas listas ou listas quase ordenadas.


Insertion Sort: O algoritmo Insertion Sort constrói uma lista ordenada, inserindo um elemento por vez em sua posição correta. Também tem complexidade O(n²), mas pode ser mais eficiente que o Bubble Sort em alguns casos.

Melhor uso: Listas pequenas ou quase ordenadas.


Selection Sort: O Selection Sort seleciona repetidamente o menor elemento e o coloca na posição correta. Ele também tem complexidade O(n²), mas é mais eficiente que o Bubble Sort, pois faz menos trocas.

Melhor uso: Pequenas listas ou quando o número de trocas deve ser minimizado.


Arrow Function e appendChild

Arrow Function: Uma Arrow Function é uma forma simplificada de escrever funções em JavaScript. Ao contrário das funções tradicionais, ela não possui o seu próprio contexto de "this", o que pode ser útil em alguns cenários.

const soma = (a, b) => a + b;

appendChild: O método appendChild permite adicionar um elemento filho a outro no DOM (Document Object Model). Ele é comumente usado para manipular a estrutura da página.

document.body.appendChild(newElement);
